# 🤖 AI Code Explainer (Offline)

An AI-powered web app that explains programming code in simple terms using a local LLM (no API key required).

---

## 🚀 Features

- 📖 Simple code explanation  
- 📊 Time complexity analysis  
- 🧠 Space complexity analysis  
- 💡 Suggestions for improvement  
- 📥 Download results  
- 🖥️ Supports multiple languages (Python, Java, C++, JavaScript)  
- 🔒 Works offline (no API key required)

---

## 🛠️ Tech Stack

- Python  
- Streamlit (Frontend UI)  
- Ollama (Local LLM)
